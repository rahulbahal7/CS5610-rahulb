DRAFTSTAT

Rating

A user can rate other user(s) that he/she is following. The user can select the number of star(s) that he would like to give the other user(out of a maximum of 20 stars).

The stars are stored with the name of the user(that the logged in user is following) in the form of FollowingSchema. That format is then embedded in an array "following":[{name,stars}, .." that is embedded in the ProjectSchema collection. The stars associated with a user that is being followed by the logged in user are stored with the name. A change in the rating is only reflected in the rating in the database.
