DRAFTSTAT

Messages

A user can view all the messages in his message inbox. The user can view the sender's name, the time at which the message was sent and the message text.

The messages are also retrieved from the MessageSchema collection in the database. From the database, the messages where the current user is the recipient are displayed in the following format:

  • Time - This is done automatically by the system. The current time is allocated to the message.
  • Sender - The logged in user is the sender.
  • Message - The message body is contained as text.
