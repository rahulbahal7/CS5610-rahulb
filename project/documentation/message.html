Messaging

A user can send messages to the user(s) that he/she is following. Upon clicking the "envelope" icon next to the user that the current user wants to send a message, a modal will be opened and the user can type in message that he wants to send by clicking the "Send Message" button.

The message functionality is implemented using the MessageSchema collection in the database. The messages is stored in the following format:

  • Sender - The logged in user is the sender.
  • Receiver - The user against which the envelope icon was clicked is the intended recipient.
  • Message - The message body is contained as text.
  • Time - This is done automatically by the system. The current time is allocated to the message.
