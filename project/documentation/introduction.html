DRAFTSTAT

INTRODUCTION

DRAFTSTAT is a web application that can be used by users to connect, share and collaborate to projects and public repositories on Github.
This application provides the users with functional features such as:

  • Search Users On Github
  • Follow Users
  • Send Messages to Followed Users
  • Rate Other Users
  • View Details of the Followed Users

    • WHY THIS PROJECT?

    This project was developed out of necessity. I have been using Github and it's repositories to share my projects. However, I always wanted Github to have a proper and well-developed social collaboration feature in it. Github Organizations come close but it still lacks a lot of features that I required from that point of view. Hence, I decided to develop this application where people could follow, rate and communicate with other "quirky" people on Github.


    DESIGN FEATURES

    The UI for this web application has been developed using the current web standards and frameworks:

    • Bootstrap
    • Responsive Design Strategy
    • Efficient Loading of Images according to the screen size
