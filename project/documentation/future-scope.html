DRAFTSTAT

Future Scope

Currently this application started off as an idea and a personal need for such an application to exist where users can collaborate effectively on Github repositories. This application has the functionalities of searching users from Github, adding users that they think are interesting, sending messages and giving other users rating.
This project can be connected to major distributed revision control systems by linking their API's. This would be a step forward in creating potentially the tool that can be used to:

  • Link users that currently use different Distributed Revision Control Systems.
  • The web application could show a more detailed view of the repositories, i.e, the file structure and source.
  • The source could be edited in an online text editor and could then be directly pushed with the changes on the original platform.
  • Creating an animation of the commits for the selected repositories.
  • Having more control for the information displayed about the user's that the current user is following.


Note: Although there are minor bugs in the system like the addition of a user more than once as a follower, however there is a potential in this viewer and can be fixed with the updates in future.
