DRAFTSTAT

SIGN-IN/REGISTER

The user is navigated to the Landing Page initially when he visits the application. On clicking the "SIGN IN / REGISTER" link at the top-right of the page, a modal is displayed which contains two sections:

SIGN IN

A registered user can enter his/her login details in the SIGN IN section of the modal. If the credentials are valid, the user will be navigated to his/her profile, otherwise an alert message will be displayed prompting the user to enter correct details.

REGISTER

A new user can use the REGISTER section of the modal to sign up for an account. The new user will be encouraged to use their Github username as their username on DRAFTSTAT. Here the user will enter their username and password. Upon confirming the passwords, the new user will be successfully registered redirected to his/her new account.
