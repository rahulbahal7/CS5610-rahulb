Search and Add Users

The user can perform a search on his/her Home Page by entering Github username in the search box. A list of potential matches returned by Github API V3 will be returned and will be displayed. The closest match would be displayed at the top with the following details:

  • Name
  • Github Handle
  • Avatar with link to Github Profile
  • Number of users following this person on Github
  • Number of users this person is following on Github
  • Number of public repositories on Github
  • List of Repositories on Github with links to their actual location

The next section contains other matches with total number of matches displayed at the top and with 30 results displayed per page. The search has pagination to handle more than 30 results. A user can type in the page number to navigate to that particular set of results, e.g: Page 1 contains results 1-30 and so on.
The search results are displayed with the following details:

  • Avatar
  • Github Username
  • Add button - To follow this particular user
