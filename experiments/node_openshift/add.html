Now, we don't want to delete all the elements and be left with nothing in the end!. So, let's see how to add a new item.
We will first need to create a simple form on the client side to take user's input. Next, an add button is displayed which will add the new object to the end of the current list of objects.
When the user clicks on the add button, the following event will be triggered on the client side:

$scope.add = function(files){
$http.post("/file", files)
.success(function(res){
$scope.files = res;
});
};


Note that this time are not considered with the index of the object as we are just going to push it to the end of the current objects. So, we just need to post and perform a push operation on the current object.

// Create a file and add it to the array

app.post("/file", function(req,res){
var object = req.body; // {fileName: "FILE", fileSize:"250KB"};
files.push(object);
res.json(files);
});
