State using Angular JS

Note

Number of characters left:

Code

var app = angular.module("noteApp", []); app.controller("noteController", function($scope) {
$scope.message = "";
$scope.left = function() {return 140 - $scope.message.length;};
$scope.clear = function() {$scope.message = "";};
$scope.save = function() {alert("Note Saved : "+$scope.message);};
});

Explaination

            This experiment is to use the state saving ability using AngularJS. The

            - ng-click="save()" 
            can be used to check the current state of the number of characters in the Text Box.

            - ng-click="clear()"
            can be used to reset the state of the Text Box.
