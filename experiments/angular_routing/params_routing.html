Parameter Routing using Angular JS

Code

The following code in the controller takes an input variable form the HTML page (songID) and we can control the variable using "$scope" of the Angular App. This way we can route information using variables in AngularJS.
sampleApp.config(['$routeProvider',
function($routeProvider) {
$routeProvider.
when('/DispSong/:songId', {
templateUrl: 'pages/show_song.html',
controller: 'SongController'
});
}]);

And then passing the parameter to the View as shown below:
$scope.song_id = $routeParams.songId;
