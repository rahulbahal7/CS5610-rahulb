Code

The followig code demonstrates how different views can be loaded in the same page by binding and invoking them using their respective controllers: 
    pageApp.config(function ($routeProvider) {
    $routeProvider        
        .when('/', {
            templateUrl: 'pages/home.html',
            controller: 'mainController'
        })
        .when('/about', {
            templateUrl: 'pages/about.html',
            controller: 'aboutController'
        })
        .when('/contact', {
            templateUrl: 'pages/contact.html',
            controller: 'contactController'
        });
});

Explanation

The AngularJS Routing is a powerful technique that can be used to implement a MVC like behaviour in web applications. The view can be loaded in a page using AngularJS Controllers and Views.
