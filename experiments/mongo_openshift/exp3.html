MongoDB and Mongoose - Retrieve and Order

About Experiment

While performing retrieval operations from the database, we often need to display data in some order. In this experiment, we will target this scenario in particular.
We will retrieve all the data from the database and will order the files based on a particular property. Here, we choose the property: file-size and will display the data in decreasing order of this property.

ng-repeat="file in files | orderBy:'size':true "
