This experiment describes the use of arrays using AngularJS. 

    var boston = {
        name: "Boston",
        country: "USA",
        temperature: "20",
    };
     var san_diego = {
        name: "San Diego",
        country: "USA",
        temperature: "60",
    };

The above objects are then rendered on the page. This experiment can be used in scenarious where multiple values are required to be manipulated and handled. 

            var cities = [boston, san_diego];

"ng-repeat" is used to iterate through the array. 

           <tbody>
              <tr ng-repeat="city in cities | orderBy:'city'">
                <td>{{city.name}}</td>
                <td>{{city.country}}</td>
                <td>{{city.temperature}}</td>
              </tr>
            </tbody>
