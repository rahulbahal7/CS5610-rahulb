Github API - Search

About Experiment

This experiment describeshow the Github API can be used to search for some information. For example, to search for users according to their Github usernames, the username needs to be supplied to the API as a GET request for which the following action needs to be performed:
$http.get("https://api.github.com/search/users?q=" + $scope.username)
