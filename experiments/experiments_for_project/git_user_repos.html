Github API - User Information

About Experiment

This experiment describes the use of the Git API to list the infomration about a user's public repositories. To List public repositories for the specified user, a GET request has to be sent in the following way:
GET /users/:username/repos
The user's public information can be retrieved using the Github API by sending a GET request in the following manner: GET /users/:username
