Github API - User Repositories

About Experiment

This experiment describes Many of the resources on the users API provide a shortcut for getting information about the currently authenticated user. If a request URL does not include a :username parameter then the response will be for the logged in user.
The user's public information can be retrieved using the Github API by sending a GET request in the following manner: GET /users/:username
